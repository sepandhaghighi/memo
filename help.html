Mnimi Game

Gameplay

Mnimi is a game of memory skill inspired by Simon electronic game. The game creates a series of lights and requires a user to repeat the sequence.

Initially, the game has four colored buttons. A round in the game consists of the device lighting up one or more buttons in a random order, after which the player must reproduce that order by pressing the buttons. As the game progresses, the number of colors, buttons to be pressed and sequence speed increases. Once the user fails, the game is over.

In Greek, mnimi means memory.
